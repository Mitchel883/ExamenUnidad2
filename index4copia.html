Estructuras de Datos

Árbol Binario

Un árbol binario es una estructura jerárquica donde cada nodo tiene como máximo dos hijos: un hijo izquierdo y un hijo derecho. Es ampliamente utilizado para representar datos jerárquicos y realizar búsquedas rápidas.

Lista Circular Enlazada

Una lista circular es una estructura de datos donde el último nodo apunta al primer nodo, formando un círculo. Esto permite recorrer la lista indefinidamente sin necesidad de saber cuándo termina.

Cola

La cola es una estructura de datos de tipo FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir. Es útil en escenarios como la gestión de procesos en sistemas operativos o en el manejo de tareas en espera.

Pila

La pila es una estructura de datos de tipo LIFO (Last In, First Out), donde el último elemento en entrar es el primero en salir. Es útil para escenarios como el deshacer operaciones en aplicaciones.

class Nodo {
  constructor(valor) {
    this.valor = valor;
    this.siguiente = null;
  }
}

class ListaCircular {
  constructor() {
    this.primero = null;
  }

  insertar(valor) {
    let nuevoNodo = new Nodo(valor);
    if (!this.primero) {
      this.primero = nuevoNodo;
      nuevoNodo.siguiente = this.primero;
    } else {
      let temp = this.primero;
      while (temp.siguiente !== this.primero) {
        temp = temp.siguiente;
      }
      temp.siguiente = nuevoNodo;
      nuevoNodo.siguiente = this.primero;
    }
  }

  recorrer() {
    if (this.primero === null) return;
    let temp = this.primero;
    do {
      console.log(temp.valor);
      temp = temp.siguiente;
    } while (temp !== this.primero);
  }
}

let lista = new ListaCircular();
lista.insertar(1);
lista.insertar(2);
lista.insertar(3);
lista.recorrer();

Código de la Cola

prueba

class Cola {
  constructor() {
    this.items = [];
  }

  enqueue(elemento) {
    this.items.push(elemento);
  }

  dequeue() {
    return this.items.shift();
  }

  verFrente() {
    return this.items[0];
  }

  estaVacia() {
    return this.items.length === 0;
  }
}

let cola = new Cola();
cola.enqueue(1);
cola.enqueue(2);
cola.enqueue(3);
console.log(cola.dequeue());  // 1
console.log(cola.verFrente());  // 2

class Pila {
  constructor() {
    this.items = [];
  }

  push(elemento) {
    this.items.push(elemento);
  }

  pop() {
    return this.items.pop();
  }

  verCima() {
    return this.items[this.items.length - 1];
  }

  estaVacia() {
    return this.items.length === 0;
  }
}

let pila = new Pila();
pila.push(1);
pila.push(2);
pila.push(3);
console.log(pila.pop());  // 3
console.log(pila.verCima());  // 2
